About PUNK API

Get Beers: Gets beers from the api, you can apply several filters using url paramaters, the available options are listed below. https://api.punkapi.com/v2/beers

Get a Single Beer: https://api.punkapi.com/v2/beers/1

Get Random Beer: https://api.punkapi.com/v2/beers/random

Response Format: json
